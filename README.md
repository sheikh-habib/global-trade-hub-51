Global Trade Center Frontend Interface
Welcome to the GlobalTrade Pro frontend repository. This is a high-performance, responsive web application designed for real-time global asset trading. The interface is built to handle live data streams with a focus on speed, accessibility, and modern aesthetics.

Tech Stack
HTML5: Semantic structure.

Tailwind CSS: Utility-first CSS framework for rapid UI development and consistent design systems.

JavaScript : Powers the client-side logic and real-time DOM updates.

Architecture & Backend Integration
This repository contains the Client-Side code only. The application follows a decoupled architecture:

Frontend: This repo (Static hosting/CDN).

Backend: A separate C# (.NET Core) repository handling authentication, trade execution, and database management via a RESTful API and WebSockets.

Features
Real-time Tickers: Live price updates.

Global Market Access: Support for multiple currencies and localized time zones.

Responsive Trading View: Fully optimized for mobile, tablet, and desktop screens using Tailwind's responsive breakpoints.
